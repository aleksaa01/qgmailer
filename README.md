# qgmailer
Currently in development, qgmailer will be a cross-platform desktop application for your Gmail account.

#### Dependencies:
``` 
    PySide2 - 5.12
    QDarkStyle - 2.6.5
    google-api-python-client - 1.7.7
    mail-parser - 3.7.1
    google_auth_oauthlib - 0.2.0
```
