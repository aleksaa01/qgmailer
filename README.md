# qgmailer
QGmailer is cross-platform desktop application for your Gmail account.
Find out more on official website: http://qgmailer.herokuapp.com/

#### Dependencies:
``` 
    PySide2 - 5.12
    QDarkStyle - 2.6.5
    google-api-python-client - 1.7.7
    mail-parser - 3.7.1
    google_auth_oauthlib - 0.2.0
```
